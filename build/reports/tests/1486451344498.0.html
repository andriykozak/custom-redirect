a) if you close it without any actions and clicks Save - the Title and Access from actual package will not be changed;
b) if you selects more modules manually or selects other package from drop-down and then Save it - the changes will apply to the site.

Actions Algorithm when get request for new site:
1) Notify Manager about the request
2) After Manager confirms it, to setup a new dealer you need to know:
   - what environment (ixloo or secure.idealersale etc.) - ask person that requested site
   - package - ask person that requested site
   - domain name to setup (if not specified in the email, ask person that requested site)
   - desired color scheme or website design (if not specified in the email, ask person that requested site)
3) add task to redmine with all information
4) create a domain on the environment
5) clone design (depend on what design was requested)
6) for any website customization such as: create a logo, custom banners etc. talk to Manager first.

How to create new website:
1. Go to ixloo.com Admin » Dealers (Root Environment site)
   - Filter by “NOT ASSIGNED SITES” and select OPEN website from the list:
a. If you have to clone design:
   - add new dealer login, pass., system name, add new domain name, select site type and Save
   - Clone required new design, after it - change package to the one you need
b. If you make design manually:
   - add new dealer login, pass., system name, select packages, add new domain name, select site type, use presets (if needed) and Save.
2. Open created website dms
   - First open each page of each module and delete previous info – “clear everything”(Important: Inventory, Leads, Scheduled Tasks, Testimonials, News, Dealer’s info, MAP meta info)
   - Go to users and create new with logins and pass. and recheck their access*
   - Go to import and import vehicles to the proper users (don't forget to select copy images to server checkbox)
   - Go to settings (Dealership, Website) - fill here all info you have,
   - If website had Auto Reports go to Auto Reports and turn on Carfax or Autocheck, then go to 3rd Party Integration and fill id/ pass for required service
   - Fill all dms info dealers had before (news, testimonials, canned images, if needed each-a-word templates, leads)
   - Go to MAP - create all pages (don't forget to recheck meta Title and description of each page)
   - Open website dws and recheck if all pages and links works correct
   - Check if website suppose to have feeds (AT, CC) or any scheduled imports/exports - create them on new website but don't activate before website goes live (don't forget include description to tasks – who and when set it up, to which user it’s assigned to and which mapping should be applied)
   - Allow to all subusers to manage inventory (if needed) - login under each users login and pass to recheck them and dms access and to be sure that all User'll be able to see inventory.
   - Go to Settings>Website>Leads Email – and set emails for Forms (if site has Custom forms – set in “Leads Emails Custom” as well)
   - If not USA cars dealer (Truck, Africa etc.) – check copyright Override settings (might be applied with presets)
   3. After converting website to live
   - If needed set up emails for dealers on root environment site (ixloo.com Admin » Email Management);
   - Activate feeds and tasks on v9
   - Check if all ftp and services received the newly sent exports
   - Follow up this website for couple days to see if inventory updated on 3rd party services ( ex. CC and AT).
* we don't give to dealers access higher then Manager, to the default manager access we also select if its possble according to website package:
